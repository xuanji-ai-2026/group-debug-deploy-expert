### 1.2 技术架构

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 客户端 | Vue3/React/Kotlin/Swift | 前后端分离，多端支持 |
| 网关层 | Spring Cloud Gateway | 统一入口，路由转发 |
| 服务层 | Spring Boot 3.x | 微服务架构 |
| 数据层 | MySQL 8.0 + Redis 7 + MongoDB | 关系型+缓存+文档 |
| 消息队列 | RabbitMQ | 异步处理，解耦 |
| AI服务 | 火山引擎/百度文心/阿里千帆 | 智能分析，话术生成 |
| 部署 | Docker + Kubernetes | 容器化部署 |

## 2. 服务划分

### 2.1 核心服务

#### bx-user（用户服务）
- 用户认证和授权
- 用户信息管理
- 角色权限管理
- 登录日志

#### bx-tenant（租户服务）
- 多租户管理
- 租户配置
- 租户隔离

#### bx-lead（商机服务）
- 商机CRUD
- 商机跟进
- 商机分配
- 商机统计分析

#### bx-content（内容服务）
- 内容发布
- 内容审核
- 内容推荐

#### bx-risk（风控服务）
- 风险评估
- 规则引擎
- 异常检测

#### bx-billing（计费服务）
- 订单管理
- 账单管理
- 支付对接
- 余额管理

#### bx-ai（AI服务）
- 模型调度
- 话术生成
- 商机评分
- 意图识别

#### bx-message（消息服务）
- 消息推送
- 通知管理
- WebSocket

#### bx-storage（存储服务）
- 文件上传
- 文件管理
- OSS对接

### 2.2 支撑服务

#### bx-gateway（网关服务）
- 路由转发
- 负载均衡
- 限流熔断
- 统一认证

#### bx-common（公共模块）
- 公共实体
- 工具类
- 全局异常
- 统一响应

#### bx-monitor（监控服务）
- 系统监控
- 性能监控
- 告警通知

#### bx-schedule（调度服务）
- 定时任务
- 任务调度

#### bx-system（系统服务）
- 系统配置
- 字典管理
- 日志管理

## 3. 数据架构

### 3.1 数据库设计

#### MySQL（关系型数据）
- 用户表
- 商机表
- 订单表
- 系统配置表

#### Redis（缓存）
- 用户Session
- 热点数据缓存
- 分布式锁
- 计数器

#### MongoDB（文档数据）
- 跟进记录
- 操作日志
- AI分析结果

#### ElasticSearch（搜索引擎）
- 商机全文搜索
- 日志分析

### 3.2 缓存策略

| 数据类型 | 缓存方式 | 过期时间 |
|----------|----------|----------|
| 用户信息 | Redis | 1小时 |
| 商机详情 | Redis | 30分钟 |
| 系统配置 | Redis | 24小时 |
| 热点数据 | Redis | 5分钟 |

## 4. 接口设计

### 4.1 RESTful API

```
GET    /api/v1/leads           # 获取商机列表
POST   /api/v1/leads           # 创建商机
GET    /api/v1/leads/{id}      # 获取商机详情
PUT    /api/v1/leads/{id}      # 更新商机
DELETE /api/v1/leads/{id}      # 删除商机
```

### 4.2 WebSocket
