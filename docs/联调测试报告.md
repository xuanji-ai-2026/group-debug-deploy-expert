## 💡 建议解决方案

### 方案A：环境准备（推荐）
**步骤**:
1. 安装JDK 17+
   ```bash
   sudo apt update
   sudo apt install -y openjdk-17-jdk
   ```

2. 安装Maven 3.8+
   ```bash
   sudo apt install -y maven
   ```

3. 安装Docker和Docker Compose
   ```bash
   sudo apt install -y docker.io docker-compose
   ```

4. 启动基础设施服务
   ```bash
   cd /workspace/projects/beijixing-ai/deploy/docker
   docker-compose -f docker-compose.dev.yml up -d mysql redis rabbitmq elasticsearch mongodb
   ```

5. 编译打包后端服务
   ```bash
   cd /workspace/projects/beijixing-ai/backend
   mvn clean package -DskipTests
   ```

6. 构建Docker镜像
   ```bash
   cd /workspace/projects/beijixing-ai/deploy/docker
   ./deploy.sh build
   ```

7. 启动所有服务
   ```bash
   docker-compose -f docker-compose.dev.yml up -d
   ```

### 方案B：使用预配置环境
**建议**:
- 将测试环境迁移到已配置好运行时的服务器
- 使用CI/CD流水线自动构建和部署
